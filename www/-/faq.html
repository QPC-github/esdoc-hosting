What is ESDoc?

Documentation generator for JavaScript(ES6).
For more information is esdoc.org

Zero config hosting(without esdoc.json)

If your repository satisfy the following conditions, enable zero config hosting(without esdoc.json).
- ES6 codes are exist in /src directory
- /package.json is exist

Documentation Coverage Badge

You can use documentation coverage badge.

Support ES7

For now, does not support ES7 in ESDoc main stream.

However experimentally support with ESDoc ES7 Plugin

Can not use property in config

- scripts
- styles

And plugins is limited used.
- esdoc-es7-plugin
- esdoc-importpath-plugin
